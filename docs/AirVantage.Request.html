    ex_vantage v0.0.1 AirVantage.Request View Source

    A module to build and execute requests to AirVantage API. The request does not happen until its passed to make_request/1. At a minimum, a request must have the endpoint and method specified to be valid.

    put_method(request, method)

    Specifies a method to use for the request. Accepts any of the standard HTTP methods as atoms, that is :get, :post, :put or :delete.

    put_param(request, key, value)

    Specify a single param to be included in the request.

    put_params(request, new_params)

    Specifies the parameters to be used for the request. If the request is a :post request, these are encoded in the request body. Otherwise, they are encoded in the URL. Calling this function multiple times will merge, not replace, the params currently specified.
